Pro tip: Export the mark as an SVG favicon (favicon.svg) — modern browsers prefer vector and it stays razor-sharp on every display density including Retina. Use the navy background (#0c0f1e) baked into the SVG so it looks intentional against any browser chrome, not transparent. Set theme-color to #0c0f1e so the browser address bar on mobile matches the brand.
